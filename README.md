
# Tasik News Auto AI

Portal berita otomatis untuk GitHub Pages.

## Fitur
- Auto ambil berita dari NewsAPI
- Generate artikel HTML otomatis
- Gambar random tiap artikel
- Rewrite sederhana agar lebih unik
- Multi bahasa via Google Translate
- Auto update via GitHub Actions

## Cara Pakai
1. Upload semua file ke repository GitHub
2. Aktifkan GitHub Pages (Deploy from branch main)
3. Masuk ke tab Actions
4. Jalankan workflow `TasikNewsAuto`

Website akan otomatis update berita.
